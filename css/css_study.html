CSS 단위

글자 크기를 명시하지 않을 경우 16px가 기본 값

px

해상도에 따라 상대적으로 달라지는 기본 단위입니다.

font-size: 12px;

px

부모 요소의 글자 크기를 100% 기준으로 계산한 % 단위입니다.

font-size: 120%;

em

부모 요소의 글자 100% 기준으로 계산한 크기의 100분의 1단위

font-size: 1.5em;

rem

부모 요소의 글자 100% 기준으로 계산한 크기의 100분의 1단위

font-size: 0.8rem;

rem

최상의 요소의 크기를 100% 기준으로 100분의 1단위한것 즉 root, html 요소를 뜻하므로,
html에서 지정한 글자 크기가 1rem이 됩니다.

font-size: 3vw;

vh

vh는 뷰포트 높이 값의 100분의 1단위

font-size: 2vh;
