함수

함수를 사용하려면 함수를 호출하고자 하는 범위 내에서 함수를 정의해야만 합니다

01. 선언적 함수

'선언적 함수'는 함수를 호출할 때 함수의 앞 또는 뒤에서 호출 가능합니다. 

{
    function func(){
        document.write("1번. 함수가 실행되었습니다.")
    }
    func();
}
결과보기

1번. 함수가 실행되었습니다.

1번. 함수가 실행되었습니다.

02. 익명함수

익명 함수는 변수에 함수 데이터를 저장하여 변수를 마치 함수처럼 사용 할 수 있도록 만들어 줍니다. 

{
    const func = function(){
        document.write("2번. 함수가 실행되었습니다.")
    }
    func();
}
결과보기

2번. 함수가 실행되었습니다.

03. 매개변수함수

함수를 호출할 때 전달하는 변수를 매개변수라고 하며, return을 이용해 함수의 처리된 결과를 반환시켜 출력시킵니다. 

{
    function func(str){ //함수 안에 넣었다고해서 파라미터,인자라고 불린다
        document.write(str);
    }
    func("3번. 함수가 실행되었습니다.")
}
결과보기

3번. 함수가 실행되었습니다.

04. 리턴값 함수

각 나이에 맞춰 요금을 구합니다. 

{
    function func(){
        const str = "4번. 함수가 실행되었습니다.4";
        return str;
    }
    document.write(func());
}
결과보기

4번. 함수가 실행되었습니다.

05.선언적 함수 : 화살표

화살표 함수는 function키워드를 생략하고 '='와'>'을 합쳐 코딩하며 항상 익명 함수 형식으로 표현합니다. 

{
    func = () => document.write("5번. 함수가 실행되었습니다.");
    func();
}
결과보기

5번. 함수가 실행되었습니다.

06. 익명함수 : 화살표

화살표 함수는 function키워드를 생략하고 '='와'>'을 합쳐 코딩하며 항상 익명 함수 형식으로 표현합니다.

07. 매개변수 : 화살표

화살표 함수는 function키워드를 생략하고 '='와'>'을 합쳐 코딩하며 항상 익명 함수 형식으로 표현합니다. 

{
    func = str => document.write(str);
    func("7번. 함수가 실행되었습니다.");
}
결과보기

7번. 함수가 실행되었습니다.

08. 리턴값 함수 : 화살표

화살표 함수는 function키워드를 생략하고 '='와'>'을 합쳐 코딩하며 항상 익명 함수 형식으로 표현합니다. 

{
    func = () => {
        const str = "8번. 함수가 실행되었습니다.";
        return str;
    }
    document.write(func())
}
