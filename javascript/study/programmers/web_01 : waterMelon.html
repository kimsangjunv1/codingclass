나누어 떨어지는 숫자 배열!

for과 if를 통한 방법입니다!

1. result 변수를 만들어줍니다
2. for를 통해 i는 1로 설정한뒤 i가 n의 이하가 될때까지 i를 1씩 더해주고
3. 만약 i가 2랑 나눴을때 0과 같다면
4. result에는 박을 더해주고 아닐시 result에는 수를 더해줍니다
5. 최종 result 반환 

function solution(n) {
    let result = ""
    for(let i = 1; i<=n; i++){
        if(i % 2 === 0){
            result += "박"
        }else{
            result += "수"
        }
    }
    return result
}
