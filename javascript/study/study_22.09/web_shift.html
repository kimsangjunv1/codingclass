Unshift

새로운 요소를 배열의 맨 앞쪽에 추가하고, 새로운 길이를 반환합니다. 

const arrNum1 = [100, 200, 300, 400, 500];
const arrNumUnshift = arrNum1.unshift(600);
번호 기본값 메서드 리턴값 결과값
1 [100,200,300,400,500] push(600) 100,200,300,400,500,600 100,200,300,400,500,600

shift

배열에서 첫 번째 요소를 제거하고, 제거된 요소를 반환합니다. 이 메서드는 배열의 길이를 변하게 합니다. 

const arrNum2 = [100, 200, 300, 400, 500];
const arrNumShift = arrNum2.shift(600);
번호 기본값 메서드 리턴값 결과값
1 [100,200,300,400,500] push(600) 100,200,300,400,500,600 100,200,300,400,500,600
