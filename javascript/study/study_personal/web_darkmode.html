다크모드?

다크모드란 평소 밝은 디자인이 위주였던 근래의 디자인을 기기의 수명과 사용자의 눈 건강을 위하여 고려된 어두운 디자인을 가진 모드를 의미합니다.

CSS : 적용법

사용자의 컴퓨터에 설정된 컬러 스키마에 맞춰 각 css 스타일이 적용될 수 있도록 prefers-color-scheme를 사용합니다. 

@media(prefers-color-scheme: dark){
    background-color: #12121a; /* !important는 사용자 개개인의 적용 가능 여부에 따라 사용하세요 ^^b! */
}
