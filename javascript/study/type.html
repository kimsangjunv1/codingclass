데이터 타입

데이터 타입은 변수에 저장되는 데이터의 유형으로 PRIMITIVE 데이터 타입과 object 데이터 타입으로 나눌 수 있습니다.

문자

큰 따옴표와 작은 따옴표로 표현 가능합니다..

맞는 예시 : "1" , '1'
틀린 예시 : 1

숫자

정수, 소수점, 지수를 표현 가능합니다.

맞는 예시 : 1
틀린 예시 : "1" , '1'

boolean(논리)

true와 false의 값을 표현합니다.

let temp1 = (5>4) //해설 : 5는 4보다 크다를 temp1에 삽입;
console.log(temp1) // 해설 : temp1(true)을 출력;
출력 값 : true

특수값

데이터값을 지정하지 않았을때, 객체 속성값을 지정하지 않았을때 표현되는 값

let temp1 = 10;
let temp2;
console.log(temp1); // 출력 10
console.log(temp2); // undefined

null

변수를 빈 상태로 만들거나 데이터를 저장했으나 값이 없을때 null로 반환합니다.

var obj=10;
obj=null;
console.log(obj); // null 출력

typeof

데이터 타입 조회시 typeof 명령어를 사용합니다.

let num = 10
let str = '문자'; console.log(typeof num); // number 출력 console.log(typeof str); // string 출력
