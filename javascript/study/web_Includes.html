includes()

includes() 메서드는 문자열 포함 여부를 검색하여, 불린(true, false)을 반환합니다.

"문자열".includes(검색값)
"문자열".includes(검색값, 시작값) - indexOf와 유사 
 const str1 = "javascript reference";

const currentStr1 = str1.includes("javascript"); //true : 관련 글자가 있을시 "true" 반환
const currentStr2 = str1.includes("j"); //true : 관련 글자가 있을시 "true" 반환

const currentStr3 = str1.includes("b"); //false : 관련 글자가 없을시 "false" 반환

const currentStr4 = str1.includes("reference"); //true : 관련 글자가 있을시 "true" 반환
const currentStr5 = str1.includes("reference", 1); //true : 관련 글자가 있을시 "true" 반환
const currentStr6 = str1.includes("reference", 12); //false : 12번째 자리에 있는것이 아니기 때문에 "false" 반환
