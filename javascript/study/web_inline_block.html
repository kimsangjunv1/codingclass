블록/인라인 구조...?

블록구조와 인라인구조는 각각의 특성이 있는데 이번 시간에 알아봅시다!

::블록구조
태그 한개가 화면상에 보이는 한 줄을 모두 차지하게 되며 여러 블록구조 태그를 사용할시 블록을 쌓듯이 계속 아래로 쌓여갑니다.

::인라인구조

블록구조/인라인구조 예시

간단한 예제를 통해 인라인 구조와 블록 구조에 대해 알아봅시다~~~~!!

block 사용 예시
<div 태그 : 나는 블록!>
<div 태그 : 나는 블록!>
inline 사용 예시
<span : 나는 인라인!> <span : 나는 인라인!>
예시용 HTML 코드보기
