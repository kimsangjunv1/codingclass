CSS 코드

폰트는 'GmarketSans'를 이용하고 아래의 한줄 효과 및 세줄 효과 css 코드를 통해 카드 UI를 구현합니다. 

/* 한줄 효과 */
overflow: hidden;
white-space: nowrap;
text-overflow: ellipsis;
padding-right: 20px;


/* 세줄 효과 */
overflow: hidden;
text-overflow: ellipsis;
display: -webkit-box;
-webkit-line-clamp: 3; /*줄 효과*/
-webkit-box-orient: vertical;

 /* ir 효과 */
display: block;
overflow: hidden;
font-size: 0;
line-height: 0;
text-indent: -9999px;
